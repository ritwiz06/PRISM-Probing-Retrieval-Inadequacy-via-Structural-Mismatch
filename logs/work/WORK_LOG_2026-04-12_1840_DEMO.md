# Work Log: Prompt 8 Demo App and Presentation Utilities

Timestamp: 2026-04-12 18:40 America/Phoenix

## Basic Summary

Prompt 8 added a local Streamlit demo layer for PRISM and two presentation-support CLIs. The app now lets a presenter run a query end to end, inspect parsed features, RAS scores, selected backend, evidence, final answer, backend-specific metadata, and reasoning trace. The demo reuses the verified PRISM pipeline instead of creating a separate retrieval path.

## What Was Implemented

- Added `streamlit` as a runtime dependency for the local demo app.
- Added `prism.demo.data` for demo view-model assembly, benchmark loading, curated example selection, evidence formatting, and export support.
- Added `prism.demo.app`, a Streamlit app with:
  - interactive query input,
  - demo presets,
  - benchmark slice/query browser,
  - parsed feature display through the payload,
  - RAS score table,
  - highlighted selected backend,
  - final answer panel,
  - retrieved evidence expanders,
  - backend-specific metadata table,
  - reasoning trace panel,
  - route match status when a benchmark query is selected.

- Extended `prism.app.pipeline.answer_query()` with an optional `retrievers` parameter so the demo can reuse preloaded retrievers without changing the existing CLI behavior.
- Added tests for demo view-model shape, benchmark loading across all four slices, export output shape, and report summary formatting.

## Demo App Summary

- Entry point: `prism/demo/app.py`.
- The app is fully local and deterministic once corpus/KG artifacts exist.
- The app uses cached retriever loading via Streamlit to avoid rebuilding indexes on every interaction.
- Presets cover all four route families:
  - lexical: `RFC-7231`
  - semantic: `What feels like climate anxiety?`
  - deductive: `Can a mammal fly?`
  - relational: `What bridge connects bat and vertebrate?`
- The benchmark browser loads lexical, semantic, deductive, and relational slices.

## Export And Report Utility Summary

- `python -m prism.demo.export_examples` writes `data/eval/demo_examples.json`.
- The export file contains 8 curated examples spanning BM25, Dense, KG, and Hybrid.
- Each exported example includes query, parsed features, RAS scores, selected backend, answer, evidence, reasoning trace, and route match metadata when a benchmark gold route is available.
- `python -m prism.demo.report_summary` prints a compact summary suitable for notes or slides.

## Known Limitations

- The Streamlit UI is functional and presentation-oriented, not a polished production frontend.
- Streamlit requires local port binding, so sandboxed environments may need explicit permission to run the server.
- The app does not yet render charts; it uses compact tables, metrics, JSON expanders, and evidence panels.
- Exported gold labels are present when the curated query exactly matches a benchmark query; free-form demo presets may not have gold labels.
- The answerer remains deterministic/template-based from Prompt 7 rather than fluent local LLM generation.
