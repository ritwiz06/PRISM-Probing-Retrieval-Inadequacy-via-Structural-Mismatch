## Known Limitations

- The public corpus is small and source-selected; it is not a web-scale retrieval benchmark.
- One public source currently uses a fallback snapshot because the live page was not fetched successfully.
- Fetched pages include formatting noise, navigation text, and boilerplate.
- Public deductive and relational cases still rely partly on the compact PRISM demo KG, so this is not a full public KG benchmark.
- Answer accuracy still uses normalized string matching rather than human judgment.
- Local-normalized and public-raw benchmarks are different suites, so deltas are directional rather than paired-example measurements.
