# PRISM Final Presentation

Open `presentation/index.html` in a browser for the interactive deck, or open `presentation/PRISM_final_presentation.pptx` in PowerPoint, Keynote, or Google Slides for the slide-deck version.

The PowerPoint deck has 7 slides and is generated by:

```bash
.venv/bin/python3 presentation/build_powerpoint.py
```

Regenerating the PowerPoint requires `python-pptx` in the virtual environment:

```bash
.venv/bin/python3 -m pip install python-pptx
```

The HTML deck is self-contained and does not require a dev server.

## Controls

- Right arrow or Space: next slide
- Left arrow: previous slide
- `N`: toggle speaker notes
- On-screen `Prev`, `Next`, and `Notes` buttons are available for mouse navigation

## Timing

The deck is designed for a 5-minute final presentation:

1. Title + thesis: 30 seconds
2. Problem and motivating example: 40 seconds
3. KRR insight / structural mismatch: 45 seconds
4. PRISM architecture and RAS: 55 seconds
5. Evaluation and key results: 70 seconds
6. Demo / evidence trace: 45 seconds
7. Takeaways and limitations: 35 seconds

## Course Requirement Coverage

- Problem worked on: slides 1-2
- KRR-based approach: slides 3-4
- Reasoning process behind the method: slide 3
- Main insight, contribution, and results: slides 5-7
- Team member contributions: slides 1 and 7
