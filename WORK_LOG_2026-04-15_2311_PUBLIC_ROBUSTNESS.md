## Public Robustness-Change Summary

The main production-safe change fixed substring marker matching in `prism/ras/feature_parser.py`.

Before this change:

- `dataclasses` accidentally triggered the deductive marker `class`.
- `threats` accidentally triggered the deductive marker `eats`.

After this change, marker matching uses token boundaries, so those queries no longer route to KG by accident.

Public-document enrichment was added as a companion metadata layer:

- identifiers,
- aliases,
- lead summaries,
- section-heading candidates,
- source type,
- source URL,
- fetch status.

Public-aware BM25 and route arbitration were added only for public-corpus analysis mode. They do not replace the default demo retriever or production router.
